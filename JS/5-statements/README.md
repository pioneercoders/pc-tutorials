
<h4> What is statement </h4>
<p> A Javascript statement is an instruction, it tells to JS what to do. </p>
<p>A program is nothing but the execution of sequence of one or more instructions.Quite often, it is desirable to alter or repeat the sequence of the statements in the program depending upon certain conditions.</p>

<p> Types of statements </p>
<ul>
	<li>Control flow Statement</li>
	<li>Branching Statement</li>
	<li>looping Statement</li>
</ul>

<h4>Statements</h4>
<p>Definition: in JS statement is single(multi) line code which ends with semicolon(;).</p>
<h4>Control Flow Statements</h4>

<p><b>The 'continue' statement:</b></p>
<p>continue statement terminates the current iteration of a loop but the loop will continue with other iterations. Essentially, it restarts the loop: (it doesn't reset the counter variable though, so the loop will continue)
.</p>
@CODE_START@@HTML@<!DOCTYPE html>
<html>
	<head>
		<title>continue statement Example</title>
	</head>
	<body>
		<script type="text/javascript">
			var i=0;

			for (i=0; i<=10; i++) {
				if (i==3) {
					continue;
				}
				document.write("The number is " + i)
				document.write("<br />" )
			}
		</script>
	</body>
</html>@CODE_END@

<p>There are if...else...if statements, to perform a multiway branch. However, this is not always the best answer, especially when all of the branches depend on the value of a single variable.
, we can use a switch statement which handles exactly this situation, and it does so more efficiently than repeated if...else if statements.
</p>                
<h4>switch Example</h4>
 @CODE_START@@HTML@<!DOCTYPE html>
<html>
	<head>
		<title>switch statement Example</title>
	</head>

			var grade = 'A';
			switch(grade){
			case 'A':
				document.write("Excellent");
				break;
			case 'B':
				document.write("GOOD");
				break;
			case 'C':
				document.write("AVERAGE");
				break;
			case 'D':
				document.write("BELOW AVERAGE");
				break;
			default :
				document.write("FAIL");		
			}
		</script>
	</body>
</html>@CODE_END@
